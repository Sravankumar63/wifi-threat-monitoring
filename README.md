# Real-Time WiFi Threat Monitoring using Passive Scanning

*(Academic Title: Wireless Network Scanner: Identifying Encryption and Signal Strengths)*

## Description
This project focuses on detecting and analyzing security threats in wireless networks using passive scanning techniques. It identifies hidden SSIDs, rogue access points, weak encryption, and suspicious network behavior using Python and Scapy.

## Features
- Passive WiFi packet sniffing using Scapy
- Detection of hidden SSIDs and rogue access points
- Encryption analysis (WEP, WPA, WPA2, WPA3)
- Signal strength monitoring
- Machine Learning-based threat classification
- Real-time alerts and logging

## Technologies Used
- Python
- Scapy
- Machine Learning
- Wireshark

## Workflow
1. Capture WiFi packets passively
2. Extract SSID, BSSID, encryption type
3. Analyze security risks
4. Detect anomalies using ML models
5. Generate alerts and reports

## Results
- Detected open and weak networks
- Identified rogue access points
- Generated real-time alerts
